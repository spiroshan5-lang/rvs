# Brand Design Guidelines

## 1. Brand Identity & Theme
- Industry: Interior Design / Architecture
- Vibe: Premium, Luxurious, Modern, Timeless
- Aesthetic: Dark mode default, rich and warm accents, glassmorphism, fluid animations.

## 2. Color Palette
- Primary Background: #0B0B0B (Rich Black)
- Primary Accent: #c9a86a (Gold / Bronze)
- Primary Text: #F5F5F0 (Warm Cream)
- Secondary Backgrounds: #1f1005, #3d2410, #4a2e15 (Warm Browns)

## 3. Typography
- Headings: Serif font (Playfair Display / Cinzel)
- Body: Sans-serif font (Inter / Outfit)

## 4. UI Elements & Styling
- Border Radius: Large cards rounded-[3.5rem], icons rounded-full.
- Glassmorphism: backdrop-blur-md, semi-transparent backgrounds.
- Borders: border-[#c9a86a]/20 or /30.

## 5. Animations & Interactions
- Hover: scale-110, slight translations.
- Transitions: duration-300.
